## Description ##

The following project is a example of using Selenium Webdriver with the pageobject-Design-pattern.
It is created for the modul EMQ und QUS - Entwicklungsmethoden und Qualitätssicherung.

## Requiered 

..* [Apache Maven](https://maven.apache.org/) 
..* Java JDK 1.8

## Installation

Use the following maven command:
`mvn install`
